# Frontend

Interfaz web del Sistema de Control de Transmetro.

## Tecnologias

- React
- Vite
- Tailwind CSS
- Leaflet
- Axios
- Socket.IO Client

## Pantallas

- Inicio de sesion.
- Dashboard segun el rol del usuario.
- Gestion de lineas, estaciones y buses.
- Directorio de personal.
- Mapa de recorridos y estaciones.
- Consulta de cobertura de seguridad.
- Control de capacidad y alertas.

## Ejecucion

```bash
npm install
npm run dev
```
